Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a web page's structure as a tree of objects. It allows scripts to dynamically access and modify the content, structure, and style of a webpage.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can use various methods depending on the criteria for selecting the element. Here are some of the most commonly used methods:
1. getElementById() Selects an element by its id attribute.
2. getElementsByClassName() Selects all elements with a specific class. This returns an HTMLCollection (which is similar to an array).
3. getElementsByTagName() Selects all elements with a specific tag name (like div, p, span). This also returns an HTMLCollection.
4. querySelector() Selects the first element that matches a CSS selector.
5. querySelectorAll() Selects all elements that match a CSS selector. This returns a NodeList, which can be iterated over like an array.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events from its child elements. It takes advantage of event bubbling, allowing events from child elements to propagate up to the parent.

Why it's useful:
Improves performance by reducing the number of event listeners.
Handles dynamic content effectively, as new child elements are automatically covered.
Simplifies code by managing events in one place.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM using JavaScript, you can use the following methods:

1. Manipulating Attributes
getAttribute(attribute): Retrieves the value of an attribute.
setAttribute(attribute, value): Adds or updates the value of an attribute.
removeAttribute(attribute): Removes an attribute from an element.
2. Manipulating Styles
element.style.property: Directly changes the inline CSS styles by setting a property, e.g.,
element.style.backgroundColor.
3. Manipulating Classes
element.classList.add(class): Adds a CSS class to the element.
element.classList.remove(class): Removes a class from the element.
element.classList.toggle(class): Toggles the presence of a class on the element.
These methods allow you to dynamically modify the appearance and behavior of elements in a web page.

What is the difference between innerHTML and textContent?

innerHTML: Inserts or retrieves HTML content, allowing you to include HTML tags within the text.
textContent: Inserts or retrieves only plain text, without interpreting HTML tags.

How do I handle events using the DOM?

You can attach event listeners to elements using:

addEventListener(event, handler): Attaches an event handler to an element (e.g., click, mouseover, etc.).
removeEventListener(event, handler): Removes an attached event handler.
